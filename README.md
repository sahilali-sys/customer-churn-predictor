# 📊 Customer Churn Prediction App

An end-to-end Machine Learning project that predicts whether a telecom customer is likely to churn.

Built using:

* Python
* Scikit-learn
* Pandas
* Streamlit

---

#  Project Overview

Customer churn prediction is an important business problem where companies try to identify customers who are likely to leave their service.

This project uses machine learning techniques to:

* analyze customer behavior
* predict churn probability
* generate business insights

The final model is deployed as an interactive Streamlit web application.

---

# Machine Learning Workflow

The project includes:

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Handling Imbalanced Data using SMOTE
* Model Training
* Model Evaluation
* Business Insight Visualization
* Streamlit Deployment

---

# Models Used

* Logistic Regression
* Random Forest Classifier

The Random Forest model achieved the best overall performance.

---

# Features Used

Some important customer features include:

* tenure

* Contract Type
* Internet Service
* Paperless Billing
